<b>Node Monitor for Digibyte (DGB).</b>

I am running my local DGB node for solo mining and tought I would create a script to monitor the node's activity and display various related data.
I hope I can contribute with this small project to the community and many of you will enjoy my creation.

<b>What are the current features?</b>

<u>Node & System</u> section:
DGB Version: displayes the current version of the node you are running.
CPU Usage: displays the CPU usage that your computer is utilizing running the node.
RAM Usage: displays the memory usage that your computer is utilizing running the node.
Disk Space: displays the amount of disk space used for the node's ''blocks' and 'chainstate' directories specifically.
